AnswerLinks is a plugin that will auto-suggest links to definitions on Answers.com for words in your blog posts.

== Installation ==

1. Download the plugin archive and expand it (you've likely already done this).
2. Put the 'anserlinks' folder into your wp-content/plugins/ directory. Afterward, you should have a folder structure like this: wp-content/plugins/answerlinks/(a bunch of files).
3. Go to the Plugins page in your WordPress Administration area and click 'Activate' for AnswerLinks.
4. If you are using a version of WP prior to 2.1, upload the included prototype.js to your wp-includes/js/ directory


== Usage ==

While authoring a post in WordPress, click the AnswerLinks button (the A in the blue circle in the rich text editor) to check for suggested links. Then choose to link or not link each suggestion and hit OK to apply your choices and return to your blog post.


== Known Issues ==

AnswerLinks does not currently work properly with Opera and is not enabled when using an Opera browser.
